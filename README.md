# Optical Biology Conference 2026 — GitHub Pages Site

This repo contains a static website for the Optical Biology Conference 2026.

## Files
- `index.html` — Home
- `2026.html` — This year's info (Trondheim / Kavli / NTNU)
- `2025.html` — 2025 recap (Leuven)
- `2024.html` — 2024 recap (Lisbon)
- `sponsors.html` — Sponsorship packages & budget
- `contact.html` — Contact & organising committee
- `style.css` — Shared stylesheet
- `script.js` — Small JS for dropdown and scroll reveal
- `img/` — folder for logos & photos (not included)

## How to deploy
1. Create a new GitHub repo (e.g. `optical-biology-conference-2026`) and push these files to the `main` branch.
2. Upload your images into the `img/` folder (names used in the HTML: `obc_logo.png`, `group_2025.jpg`, `group_2024.jpg`, `ucl_logo.png`, `wellcome_logo.png`, `vib_logo.png`, `kavli_logo.png`, `ntnu_logo.png`).
3. Go to **Settings → Pages** (or **Main settings → Pages**) and set the source to `main` branch / root. Save.
4. Your site will be published at `https://<yourusername>.github.io/<repo-name>/`.

## Notes

- If you want a custom domain, add it in Pages settings and create a `CNAME` file if required.
